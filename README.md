# Majestic Family Bot

[![Python](https://img.shields.io/badge/python-3.9+-blue.svg)](https://python.org)
[![aiogram](https://img.shields.io/badge/aiogram-2.25-green.svg)](https://github.com/aiogram/aiogram)

Telegram-бот для управления игровым кланом (семьей) с системой ролей, варнов, каптов и рангов.

## 🚀 Возможности

- **Ролевая система**: Leader, Vice, Elite, Member, Recruit
- **Приглашения**: Генерация уникальных ссылок для вступления
- **Варны**: До 3 предупреждений, после чего автоматический кик
- **Выдача фула**: Администраторы могут выдавать оружие, броню и паттерны
- **Система каптов**: 5 атакующих и 3 защищающих капта (без реализации битвы, просто учёт)
- **Ранги 1-10**: Повышение и понижение рангов
- **Машины по рангам**: Каждый ранг получает доступ к определённому списку машин
- **Админ-панель**: Управление всеми функциями через удобные кнопки
